Chapter 3: Coordinate Geometry

Class 9 Mathematics Updated: May 15, 2025

Introduction

Coordinate Geometry combines algebra with geometry by using a coordinate system to describe points and shapes in a plane. This chapter introduces the Cartesian coordinate system and its applications in solving geometric problems using algebraic methods.

Key Concepts

Cartesian Coordinate System

  • Two perpendicular lines (axes) intersecting at origin (0,0)
  • Horizontal line is x-axis, vertical line is y-axis
  • The plane is divided into four quadrants
  • Any point can be located using ordered pair (x,y)
  • Signs of coordinates in different quadrants:
    • Quadrant I: (+,+)
    • Quadrant II: (-,+)
    • Quadrant III: (-,-)
    • Quadrant IV: (+,-)

Distance Formula

  • Distance between points (x₁,y₁) and (x₂,y₂)
  • Formula: d = √[(x₂-x₁)² + (y₂-y₁)²]
  • Based on Pythagorean theorem
  • Used to find:
    • Length of line segments
    • Perimeter of polygons
    • Whether points are collinear

Section Formula

  • Coordinates of point dividing line segment in ratio m:n
  • x-coordinate: (mx₂ + nx₁)/(m+n)
  • y-coordinate: (my₂ + ny₁)/(m+n)
  • Special case: Midpoint formula when m=n=1
  • Midpoint coordinates: ((x₁+x₂)/2, (y₁+y₂)/2)

Area of Triangle

  • Area using coordinates of vertices
  • Formula: ½|x₁(y₂-y₃) + x₂(y₃-y₁) + x₃(y₁-y₂)|
  • Positive value gives actual area
  • Can be used to check if points are collinear

Important Questions and Solutions

Question 1

Find the distance between points A(3,4) and B(-2,1).

Solution:

Using the distance formula:

  1. d = √[(x₂-x₁)² + (y₂-y₁)²]
  2. d = √[(-2-3)² + (1-4)²]
  3. d = √[(-5)² + (-3)²]
  4. d = √[25 + 9]
  5. d = √34

Therefore, the distance between A and B is √34 units.

Question 2

Find the coordinates of the point which divides the line segment joining A(2,-3) and B(4,7) in the ratio 2:3.

Solution:

Using the section formula:

  1. For ratio m:n = 2:3
  2. x-coordinate = (mx₂ + nx₁)/(m+n) = (2×4 + 3×2)/(2+3) = 14/5
  3. y-coordinate = (my₂ + ny₁)/(m+n) = (2×7 + 3×(-3))/(2+3) = 5/5 = 1

Therefore, the required point is (2.8, 1).

Key Formulas

Distance Formula

d = √[(x₂-x₁)² + (y₂-y₁)²]

Section Formula

x = (mx₂ + nx₁)/(m+n)
y = (my₂ + ny₁)/(m+n)

Midpoint Formula

x = (x₁+x₂)/2
y = (y₁+y₂)/2

Area of Triangle

A = ½|x₁(y₂-y₃) + x₂(y₃-y₁) + x₃(y₁-y₂)|

Common Misconceptions

Order of Coordinates

Misconception: The order of x and y coordinates doesn't matter.

Correction: The order is crucial - (x,y) and (y,x) represent different points unless x=y.

Distance Formula Application

Misconception: Distance formula gives negative results sometimes.

Correction: Distance is always positive as it involves square root of sum of squares.

Area Formula Sign

Misconception: The area formula can give negative results.

Correction: We take the absolute value in the area formula to get the actual area.
