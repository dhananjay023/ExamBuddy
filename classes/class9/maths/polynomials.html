Chapter 2: Polynomials

Class 9 Mathematics Updated: May 15, 2025

Introduction

Polynomials are algebraic expressions that consist of variables and coefficients. This chapter introduces you to different types of polynomials, their properties, and operations that can be performed on them. Understanding polynomials is crucial as they form the foundation for many advanced mathematical concepts.

Key Concepts

What is a Polynomial?

  • An algebraic expression in which variables are combined with coefficients using only addition, subtraction, and multiplication
  • The exponents of variables must be whole numbers
  • Examples: x² + 2x + 1, 3y³ - 2y + 5, 2z
  • Counter-examples: 1/x (negative exponent), √x (fractional exponent)

Types of Polynomials

  • Monomial: Single term (e.g., 3x², 5)
  • Binomial: Two terms (e.g., x + 2, y² - 3)
  • Trinomial: Three terms (e.g., x² + 2x + 1)
  • Based on degree:
    • Linear: Degree 1 (e.g., 2x + 3)
    • Quadratic: Degree 2 (e.g., x² - 4x + 4)
    • Cubic: Degree 3 (e.g., x³ + 2x² - x + 1)

Zeroes of a Polynomial

  • A value of x for which p(x) = 0
  • Also called roots of the polynomial
  • A polynomial of degree n has at most n zeroes
  • Methods to find zeroes:
    • Factorization
    • Factor theorem
    • Remainder theorem

Important Identities

  • (a + b)² = a² + 2ab + b²
  • (a - b)² = a² - 2ab + b²
  • (a + b)(a - b) = a² - b²
  • (x + a)(x + b) = x² + (a+b)x + ab

Important Questions and Solutions

Question 1

Find the value of k if (x - 1) is a factor of x³ - 3x² + kx - 6.

Solution:

Using the factor theorem:

  1. If (x - 1) is a factor, then x = 1 should satisfy the polynomial
  2. Put x = 1 in the polynomial: (1)³ - 3(1)² + k(1) - 6 = 0
  3. 1 - 3 + k - 6 = 0
  4. k - 8 = 0
  5. k = 8

Therefore, the value of k is 8.

Question 2

Factorize: x² - 5x + 6

Solution:

Let's solve this step by step:

  1. This is a quadratic polynomial
  2. We need to find two numbers whose:
    • Sum is -5 (coefficient of x)
    • Product is 6 (constant term)
  3. The numbers are -2 and -3
  4. Therefore, x² - 5x + 6 = (x - 2)(x - 3)

Key Formulas

Square of Sum

(a + b)² = a² + 2ab + b²

Square of Difference

(a - b)² = a² - 2ab + b²

Difference of Squares

(a + b)(a - b) = a² - b²

Cubic Identity

a³ + b³ = (a + b)(a² - ab + b²)

Common Misconceptions

All Polynomials Have Real Roots

Misconception: Every polynomial has real roots.

Correction: Not all polynomials have real roots. For example, x² + 1 has no real roots.

Degree of Zero Polynomial

Misconception: The zero polynomial has degree zero.

Correction: The degree of the zero polynomial is not defined.

Number of Terms vs Degree

Misconception: The number of terms in a polynomial determines its degree.

Correction: The degree of a polynomial is determined by the highest power of the variable, not the number of terms.
