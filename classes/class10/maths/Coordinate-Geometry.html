Chapter 7: Coordinate Geometry

Class 10 Mathematics Updated: May 15, 2025

Introduction

Coordinate Geometry combines algebra with geometry by using a coordinate system to describe geometric shapes and their properties. This chapter introduces fundamental concepts like distance formula, section formula, and area of triangles in the coordinate plane, providing essential tools for solving geometric problems algebraically.

Key Concepts

Distance Formula

  • Distance between points (x₁, y₁) and (x₂, y₂):
    d = √[(x₂ - x₁)² + (y₂ - y₁)²]
  • Applications:
    • Finding perimeter of polygons
    • Verifying geometric properties
    • Checking if points are collinear

Section Formula

  • Internal division (m:n ratio):
    x = (mx₂ + nx₁)/(m + n)
    y = (my₂ + ny₁)/(m + n)
  • External division (m:n ratio):
    x = (mx₂ - nx₁)/(m - n)
    y = (my₂ - ny₁)/(m - n)
  • Special cases:
    • Midpoint formula (m = n = 1)
    • Centroid of triangle (m = n = 1)

Area of Triangle

  • Using coordinates (x₁, y₁), (x₂, y₂), (x₃, y₃):
    Area = ½|x₁(y₂ - y₃) + x₂(y₃ - y₁) + x₃(y₁ - y₂)|
  • Properties:
    • Area is always positive
    • Order of points doesn't matter
    • Zero area indicates collinear points

Important Questions and Solutions

Question 1

Find the ratio in which the line segment joining the points A(6, -4) and B(-3, 5) is divided by the y-axis.

Solution:

Let's solve this step by step:

  1. The y-axis has equation x = 0
  2. Let P(0, y) be the point where the line segment AB intersects y-axis
  3. Using section formula:
    • If P divides AB in ratio m:n, then:
    • 0 = (m(-3) + n(6))/(m + n)
    • 0 = -3m + 6n
    • 3m = 6n
    • m:n = 2:1
  4. Therefore, y-axis divides AB in ratio 2:1

Question 2

Show that the points A(2, -2), B(14, 10), and C(2, 10) are the vertices of a right triangle.

Solution:

To prove that triangle ABC is right-angled, we'll use the distance formula and verify using the Pythagorean theorem.

  1. Find AB:
    • AB = √[(14-2)² + (10+2)²]
    • AB = √[144 + 144]
    • AB = √288 = 12√2
  2. Find BC:
    • BC = √[(2-14)² + (10-10)²]
    • BC = √144 = 12
  3. Find AC:
    • AC = √[(2-2)² + (10+2)²]
    • AC = √144 = 12
  4. Check Pythagorean theorem:
    • AC² + BC² = 144 + 144 = 288
    • AB² = 288
    • Therefore, AC² + BC² = AB²

Since the Pythagorean theorem is satisfied, triangle ABC is right-angled at C.

Key Formulas to Remember

Distance Formula

d = √[(x₂ - x₁)² + (y₂ - y₁)²]

Section Formula

Internal division (m:n)
x = (mx₂ + nx₁)/(m + n)
y = (my₂ + ny₁)/(m + n)

Area Formula

Area = ½|x₁(y₂ - y₃) + x₂(y₃ - y₁) + x₃(y₁ - y₂)|
